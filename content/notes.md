## Shortcuts for npm batch installation packages

_2023/04/16_

I only know now

Shortcuts for npm batch installation packages

```bash
npm/pnpm i -D @types/react @types/react-dom @types/node
// or
npm/pnpm i -D @types/{react,react-dom,node}@latest
```
